CENTINELA OSINT - Spanish-Language Threat Intelligence Monitor
Author: Sebastian Guerrero
Built with: Python & Claude AI
Platform: Google Colab

Purpose:

Monitors simulated Spanish-language threat feeds (dark web, forums, Telegram channels, clandestine marketplaces) and uses Claude to extract structyred threat intelligence - TTPs, severity, IOCs and analyst recommendations - bridging the gap in English-centric SOC tooling.
